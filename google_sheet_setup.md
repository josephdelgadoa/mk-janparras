# Google Sheet Setup: "Vallarta Vows Leads"

## Step 1: Create the Sheet
1. Go to Google Sheets.
2. Create a new sheet.
3. Name it **"Vallarta Vows Leads"**.
4. Rename the first tab (bottom) to **"Leads"**.

## Step 2: Add Headers
Copy the following line (CSV format) and paste it into cell **A1** of your new sheet. After pasting, if it doesn't split automatically, go to **Data > Split text to columns**.

```csv
Timestamp,Partner 1 First Name,Partner 1 Last Name,Partner 1 Gender,Partner 2 First Name,Partner 2 Last Name,Partner 2 Gender,Email Address,Phone Number,Wedding Date,Guest Count,Budget Range,Location Preference,Referral Source,Services Needed,Message
```

## Step 3: Formatting (Optional)
- **Bold** the header row (Row 1).
- **Freeze** the first row (View > Freeze > 1 row).
- **Resize** columns to fit content.
